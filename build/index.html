Usage

Note: for PNG usage, please consult view the source of those sample files. They are provided only for legacy situations.

<i class="fsico-icon-name"></i>

That's it. Scale however you want. Include the CSS file on your page.

You can also choose to use the fillTypes.js script - simply include in on your page, and then you can do:

<fillType name="filltype-here"></fillType>

Fill type names are normalized, and I have gone to great pains to guess as well as possible what you meant - spelling matters, capitalization does not.
